O que é CSS

CSS (Cascading Style Sheets) é uma linguagem usada para descrever a apresentação de um documento HTML. Com CSS você pode controlar cores, fontes, espaçamento, layout e muito mais.

  • História do CSS: desenvolvido em 1996 para separar conteúdo e estilo
  • CSS3: versão atual com suporte a animações, responsividade, variáveis e seletores avançados
  • Sintaxe: seletor { propriedade: valor; }

Exemplo básico de CSS

body {

background-color: #f0f0f0; font-family: Arial; color: #333; } h1 { color: #0077cc; text-align: center; }

Vantagens do CSS

  • Separar conteúdo e estilo
  • Reduzir redundância
  • Reutilização de estilos
  • Melhor controle visual
  • Responsividade com media queries

Desvantagens do CSS

  • Pode ser complexo em grandes projetos
  • Difícil de depurar estilos conflitantes
  • Compatibilidade entre navegadores
