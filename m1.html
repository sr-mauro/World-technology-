O que é HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar páginas na web. Ele estrutura o conteúdo, definindo elementos como títulos, parágrafos, listas, links, imagens e muito mais.

  • História do HTML: criado por Tim Berners-Lee em 1991
  • HTML5: versão mais atual com suporte a vídeo, áudio, semântica e responsividade
  • Tags comuns: <h1> até <h6>, <p>, <a>, <img>, <div>, <section>

Exemplo básico de HTML

<!DOCTYPE html>
<html lang="pt">
<head>
  <meta charset="UTF-8">
  <title>Meu Primeiro Site</title>
</head>
<body>
  <h1>Bem-vindo ao mundo do HTML!</h1>
  <p>HTML é fácil e poderoso.</p>
</body>
</html>

Vantagens do HTML

  • Fácil de aprender e usar
  • Suporte universal: funciona em todos os navegadores
  • Gratuito e aberto
  • Integração com CSS e JavaScript
  • Estrutura clara de conteúdo

Desvantagens do HTML

  • Limitado para lógica de programação (sem condições/loops)
  • Layout limitado sem CSS
  • Não é interativo sozinho – precisa de JavaScript
  • Repetitivo em projetos grandes

Principais Tags HTML

HTML utiliza uma série de tags que definem a estrutura de uma página web. Aqui estão algumas das mais comuns:

  • <html>: Define o início e o fim de um documento HTML.
  • <head>: Contém metadados sobre o documento (como título, links para CSS, etc.).
  • <body>: Contém o conteúdo visível da página.
  • <h1>, <h2>, <h3>: Usadas para títulos e subtítulos.
  • <p>: Define um parágrafo de texto.
  • <a>: Define um link para outra página ou recurso.
  • <img>: Exibe uma imagem na página.
  • <ul>, <ol>: Define listas não ordenadas e ordenadas, respectivamente.
  • <form>: Define um formulário HTML.

Atributos em HTML

Além das tags, HTML usa atributos para modificar o comportamento e a aparência dos elementos. Alguns dos atributos mais comuns incluem:

  • href: Usado em tags de link (<a>) para definir o destino do link.
  • src: Usado em tags de imagem (<img>) para definir o caminho da imagem.
  • alt: Usado em imagens para fornecer uma descrição alternativa da imagem.
  • class: Usado para definir uma classe para um elemento, o que permite aplicar estilos CSS a ele.
  • id: Usado para dar um identificador único a um elemento, útil para scripts e estilos.

Formulários HTML

Os formulários são essenciais para coletar dados dos usuários. Eles podem ser usados para pesquisas, login, e-commerce, entre outros. Um exemplo básico de formulário HTML:

        <form action="/submit" method="POST">
            <label for="nome">Nome:</label>
            <input type="text" id="nome" name="nome">
            <label for="email">Email:</label>
            <input type="email" id="email" name="email">
            <input type="submit" value="Enviar">
        </form>
