O que é JavaScript

JavaScript é uma linguagem de programação interpretada, usada para adicionar interatividade a páginas web. Ele pode modificar o HTML e CSS de uma página dinamicamente, tornando o site mais responsivo e funcional.

  • História do JavaScript: criado por Brendan Eich em 1995
  • ES6: versão moderna com let/const, arrow functions, classes, promises e muito mais
  • Utilizações: validação de formulários, animações, interatividade, chamadas de APIs

Exemplo básico de JavaScript

<script>

function saudacao() { alert("Olá, bem-vindo ao mundo do JavaScript!"); } saudacao();

Vantagens do JavaScript

  • Executado no navegador
  • Interatividade em tempo real
  • Grande comunidade e suporte
  • Bibliotecas e frameworks poderosos (React, Vue, etc.)

Desvantagens do JavaScript

  • Depende do navegador do usuário
  • Pode ser desabilitado
  • Erros de segurança se mal utilizado
